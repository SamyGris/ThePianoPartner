FMOD :

FMOD est une bibliothèque multiplateforme de gestion du son,L'avantage de celle-ci est qu'elle peut être utilisée au travers de nombreux langages de programmation. Elle est utilisée par un bon nombre de personnes meme si il faut parfois "creuser" dans les documentations pour trouver ce qu'il nous faut. L'avantage de cette bibliothéque est le fait qu'elle réponde aux exigences du projet. En effet, Fmod est une bibliothèque trés riche avec lequel il est plus facile de jouer nos notes de piano/ ajuster le tempo ... etc . Cependant, l'installation de cette dernière sur Linux est assez compliquée puisque la plupart des gens l'utilisent sur Windows.De plus, la plupart des documentations sont en Java ce qui nous oblige à repenser les algorithmes de base pour un langage bas niveau comme le C .
